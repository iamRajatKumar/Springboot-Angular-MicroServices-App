About This Quiz App

This Quiz Application is designed to help users practice and test their knowledge on various topics through a dynamic quiz system.

🔍 Purpose of the App

- To allow users to take quizzes fetched from a backend service
- To validate answers and display whether the chosen option is correct
- To demonstrate a full-stack application using Angular and Spring Boot microservices

🛠 Tools & Technologies Used

  • Angular ─ for building the frontend UI
  • Spring Boot Microservices ─ for backend quiz and question services
  • Spring Cloud & Eureka ─ for service registry and discovery
  • API Gateway ─ to route requests to services
  • JSON Server / MySQL ─ for storing quiz and question data
  • HTML, CSS ─ for page structure and styling
